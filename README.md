# Análisis del Rendimiento de Equipos de Fútbol mediante Integración de Múltiples Fuentes de Datos

## Descripción

Este proyecto tiene como objetivo integrar, limpiar, transformar y analizar información deportiva proveniente de múltiples fuentes de datos utilizando herramientas de análisis de datos.

Se emplearon procesos ETL desarrollados en KNIME, análisis exploratorio en Python y visualizaciones interactivas en Power BI para facilitar la interpretación de indicadores relacionados con el rendimiento de equipos de fútbol.

---

- Python
- Pandas
